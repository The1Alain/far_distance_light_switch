# Far_distance_light_switch
## English 
### Material :
- NRF 24LO1
- Arduino Uno
- Arduino Nano
- Breadboard
- Mini breadboard
- 14 male to female wires
- 14 male to male wires
- 2 10 K resistors
- 2 buttons
- Double sided tape
- 2 servo motors
- 2 screws
### 3D printing Info :
The STL file posted might not work on every light switch since the shape and size varies from house to house, so before printing it is better to check if the mesurments are aligned with the shape of your switch. It is still in its prototype phase so the servo motors might not fit properly depending on the used printer.
### Assembly Info :
1. Apload the transmitter code to the arduino Nano the proceed to the wiring.
2. Apload the receiver code to the arduino Uno.
3. Screw the servo motor heads and use tape to stcik the servo motors to the holes of the printed part.
4. Use the double sided tape to stick the compents and 3D printed piece to the Wall.
5. Adjust the initial and final angles of the servo motors since the angles required to turn of a switch could vary from house to house (the code to modify is indicated on the c++ file).
### Important Info :
This is a prototype and might not work if the lights switches require more strenght than the one that a servo motor can produce. Changing the angles could in some cases fix the problem.
